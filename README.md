# open-deepsearch
open-deepsearch ( Deep Research but Open-Sourced )

## How to install in DEV environment after git clone
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install -e .
deepsearch
```
